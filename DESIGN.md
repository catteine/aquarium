# Aquarium App — Design Guide

## 전체 컨셉
손으로 그린 드로잉 느낌의 아쿠아리움. 물고기도 직접 그려서 추가하고, 어항 자체도 스케치 선으로 표현. 깔끔하고 단순한 화이트 베이스 UI.

## 색상

| 요소 | 색상 |
|------|------|
| 페이지 배경 | 흰색 (`#ffffff`) |
| 하단 툴바 배경 | 흰색 (`#ffffff`) |
| 툴바 구분선 | `border-zinc-200` |
| 어항 스케치 선 | 순수 블랙 (`#000000`) |
| 모래 바닥 fill | `rgba(195, 168, 118, 0.38)` — 옅은 모래색 |
| 물고기 추가 버튼 | `bg-indigo-500` |
| 전체 삭제 버튼 | 테두리형 (`border-zinc-300`, `text-zinc-500`) |
| 삭제 확인 버튼 | `bg-red-500` |
| 설정 버튼 | `bg-zinc-100` |

## 어항 (Canvas 렌더링)

- **물 색상 없음** — 배경(흰색)이 그대로 보임
- **외곽 테두리**: 손으로 그린 스케치 선. `roughLine()` 으로 각 변을 2회 겹쳐 그림
  - 선 두께: 2.2px, 색상: `#000000`, `lineCap: round`
  - 각 변을 ~55px 간격으로 나눠 ±2.5px 랜덤 흔들림 적용
- **모래 바닥**: 울퉁불퉁한 `quadraticCurveTo` 언덕 반복, 높이 45px
- **스케치 캐싱**: 리사이즈 시에만 오프스크린 canvas 재생성 → 매 프레임 `drawImage` stamp
- **비율**: `max-width: 900px`, `aspect-ratio: 16/9`

## 물고기

- **드로잉 방식**: 사용자가 캔버스에 직접 그림 (freehand)
- **표시 크기**: 80×60px
- **움직임**: 랜덤 속도(`±1 ~ ±2 px/frame`), 벽/모래 바운스
- **좌우 반전**: 이동 방향에 따라 자동 flip. `facingRight` 속성으로 원본 방향 보정
- **바운스 경계**: 테두리 안쪽 `BORDER_PAD=14px`, 모래 위 `SAND_HEIGHT=45px`

## 레이아웃

```
┌────────────────────────────────┐  bg-white
│                          [⚙]  │  상단 (p-3)
│                                │
│   [스케치 어항 canvas]         │  flex-1, 중앙 정렬
│                                │
│  물고기 N마리  [전체삭제] [+]  │  하단 툴바, border-t zinc-200
└────────────────────────────────┘
```

## 드로잉 모달

- 흰색 패널, `rounded-2xl`, 그림자
- 드로잉 캔버스 배경: 투명 (어항과 동일한 느낌)
- 색상 팔레트: 빨강/주황/노랑/초록/파랑/보라/흰색/검정
- 브러시 크기 슬라이더, 지우개, 초기화 버튼
- **머리 방향 토글**: 그림이 바라보는 방향(`← 왼쪽 / 오른쪽 →`) 선택 후 추가
- 캔버스에 방향 가이드 오버레이 표시

## UX 패턴

- **전체 삭제**: 2단계 확인 모달 (1단계: "물고기가 삭제됩니다." → 2단계: "정말 삭제하시겠습니까?")
- **설정 메뉴**: 우측 상단 ⚙ 버튼 → 드롭다운 (JSON 저장 / JSON 불러오기)
- **JSON 불러오기**: 아쿠아리움이 비어있을 때만 활성화
- **자동 저장**: 물고기 추가/삭제 시 LocalStorage에 자동 저장, 새로고침 시 복원
